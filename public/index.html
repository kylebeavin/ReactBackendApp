Get started

    API Endpoint

    https://smash-app-backend.herokuapp.com/api

The Smash My Trash API provides programmatic access to read SMT data. Retrieve a user, group, details, filter them, etc.

To use this API, you need to be logged in with Authorized Account.

get users

To get users you need to make a GET call to the following url :
https://smash-app-backend.herokuapp.com/api/users/

QUERY PARAMETERS

Field Type Description
_id ObjectID Document ID assigned by Mongo
user_id String User ID assigned by Google
first_name String Document ID
last_name String Document ID
search String (required) A search word to find user by name.
group String (tbd) Gets the users:
active Boolean (tbd) a boolean to filter active users
